# Elective Blocks — Блоки по выбору

Each candidate picks **two** elective blocks before the exam. Questions are drawn from the chosen blocks in addition to the [core block](../core/).

| # | Folder | Topic |
|---|--------|-------|
| 2.1 | [2.1-mathphys-numerical/](2.1-mathphys-numerical/) | Equations of Mathematical Physics and Computational Mathematics |
| 2.2 | [2.2-discrete-algebra/](2.2-discrete-algebra/) | Discrete Analysis and Foundations of Higher Algebra |
| 2.3a | [2.3a-logic-tcs/](2.3a-logic-tcs/) | Foundations of Mathematical Logic and Theoretical Computer Science |
| 2.3b | [2.3b-programming/](2.3b-programming/) | Programming Fundamentals |
| 2.4 | [2.4-convex-optimization/](2.4-convex-optimization/) | Convex Analysis, Optimization Theory and Optimal Control |
| 2.5 | [2.5-advanced-ai/](2.5-advanced-ai/) | Advanced Artificial Intelligence Methods |
| 2.6 | [2.6-it-product/](2.6-it-product/) | IT Product Management |

> Note: the source program ([../программа.pdf](../программа.pdf)) numbers two distinct blocks as «2.3». They are split here as **2.3a** (logic / theoretical CS) and **2.3b** (programming).

**How to choose:** match electives to your target кафедра. Heavy-math kafedras → 2.1, 2.2, 2.4. ML/AI kafedras → 2.4, 2.5. Engineering/Software kafedras → 2.3b. CS theory kafedras → 2.3a.
