\section{Computer Security}
\label{sec:2.3b.11-computer-security}

\begin{ticket}
OS security: access control models (DAC, MAC, RBAC); memory and execution protection (ASLR, DEP, canaries); mandatory control and isolation (SELinux, AppArmor). Malware: classification, propagation, detection (signature, behavioral, sandbox). Intrusion monitoring (syslog, auditd; event correlation; anomaly detection). Incident response: DR plans, backup strategies, post-incident review. Cloud security: shared-responsibility model, data security, IAM, multi-tenancy isolation.
\end{ticket}
