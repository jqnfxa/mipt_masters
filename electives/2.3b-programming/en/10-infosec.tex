\section{Information Security}
\label{sec:2.3b.10-infosec}

\begin{ticket}
Information security: concepts, threats, risk models, legal aspects. Cryptography: symmetric / asymmetric encryption, hash functions, digital signatures, PKI. PRNGs. Vulnerabilities and attacks: injections (SQL, XSS), overflows, RCE, phishing, DDoS. Application defense: secure development, input handling, password storage, access control. Network security: firewalls, IDS/IPS, VPN, TLS/SSL. Attacks: man-in-the-middle, spoofing, flooding.
\end{ticket}
